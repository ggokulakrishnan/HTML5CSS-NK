1. Think all the elements of HTML appearing as a box in a page
  2. Both inline and Block level elements occupy a box
  3. inline HTML element: do not occupy the fill width of the page
  4. block level HTML element: they occupy the full width of the page
  5. all the boxes has margin, padding and border
  6. block level elements have width property whereas
  7. inline level elements do not have width property
  8. In the below image, consider the element (2nd box) which is our element
  9. Padding: the inner part of a html element is called as padding
  10. margin: the outer part of html element is called as margin
  11. inline elements: they do not display vertical margin
  12. but they display left and right margin (did not understand)
  13. Margin can be defined as follows:
    margin-top: 10px;
    margin-right: 10px;
    margin-bottom: 10px;
    margin-left: 10px;

  14. if all sides of the box are of same width (which means we need to give 10px space to top, right, bottom and left
  15. then it can be written as:
    margin: 10px;

  16. There is an another way to write the margin values in single row
    margin: 0 10px 20px 30px;
    Here
    0: represents top
    10px: represents right
    20px: represents bottom
    30px: represents left

  17. padding also behaves the same instead of margin use padding with above syntax
  18. There are few other ways of representation (2 value representation
  19. margin: 0 10px;
    Here
    0: represents top and bottom and
    10px: represents left and right

  20. margin and padding (3 value representation)
  21. margin: 0 10px 20px;
    Here
    0: represents top
    10px: represents left and right and
    20px: represents bottom

  22. borders: also work the same way as margin and padding
  23. borders have three property (style width and color)
    border-style: solid; (other style: single line, double line, dotted, dashed)
    border-width: 4px;
    border-color: #333;

  24. alter option to write the border in single line
    border: solid 1px red;

  25. In border it is non mandatory to write the border in same order of representation (style, width and color)
  26. the values can be interchanged also
  27. border: solid red 1px;
  28. borders once defined can also be overridden
    border: solid 1px red;
    border-bottom: green;
    Here the order color is defined as red and then only the bottom-border is defined with green color
