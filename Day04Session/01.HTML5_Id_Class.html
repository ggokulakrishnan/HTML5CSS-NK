Id and Class

  1. < link > tag used in the head section of HTML document is used to link/connect this document to an another document
  2. link tag does not have/require a closing tag
  3. rel = relationship
  4. rel="stylesheet" this is a predefined value
  5. relationship has many values and one of the value is stylesheet
  6. href="style.css" is linking stylesheet file we need to refer for this document
  7. to give style to an HTML element we use Id and Class
  8. "id": is an unique identification and can be used only once in a HTML document
  9. in the below example, id="idSelector" is called id selector
  10. the # (hash) symbol, is used to represent an id
  11. "class": is a global identification and can be used multiple times in a HTML document
  12. in the below example, class="clsSelector1" is called class selectors
  13. the . (dot) symbol, is used to represent a class
  14. the prefix of class and id (. and #) need to applied only inside style (inline) or css (Ext. file) and not inside HTML element section

Example of using "id" attribute:

(a) id is an unique identification and can be used only once in a HTML document
(b) when defining the style they are represented with # (hash) symbol
(c) at the place of calling the style (in HTML element) the # symbol should not be used
(d) Example (while defining): #idSelector { color: red; }
(e) Example (while calling): id="idSelector" (or) <p id="idSelector"></p>

Example of using "class" attribute:

(a) class is global identifier and can be used mutiple times in a single HTML document

(a) when defining the style they are represented with . (dot) symbol

(a) at the place of calling the style (in HTML element) the . symbol should not be used

Example (while defining): .clsSelector { color:purple; }
Example (while calling): class="clsSelector" (or) <p class="clsSelector"></p>
