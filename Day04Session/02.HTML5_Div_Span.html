Div and Span

  • div is a block level element (which means it will cover the entire width of the page)
  • div, is basically a container which will hold all other html elements
  • during initial days tables were used to make page layouts, but now a days during dynamic sites div is used extensively
  • except mail templates, we use div in all the places. Mailer does not supports div, we use tables even today
  • div, is mostly used in combination with css (style sheet), java scripts etc..
  • in the example below: div id="container": where container is a class (style)
  • in this sxample we will use an external css using (link) tag and it will be placed in the (head) section
  • span, is used inline (eg) inside a paragraph tag and it is used for styling
  • in the below example we have created a (p) tag and added inline (span) tag
  • using external stylesheet we will add style to the (span) tag
  • #container span { } (-- create a new style similar to this, because our span is inside the container and make the data inside span tag to bold and font with blue in color
  • in css to bold an element we use the property: font-weight: bold; and now run it
  • an other way to bring the same style is as below:
  • #container .bold { font-weight: bold; }
  • (span class="bold") this is a span tag ...
  • an other way to bring the same style is as below:
  • div span { font-weight: bold; }
  • the third style option is not recommended because, in a page if there are multiple div and span then the same style will be applied to all of them
  • hence the best option is to use id and class format
  • interpretation: #container span { font-weight: bold; }
  • if an element is seen with an id of name (container) and which contains a HTML element (span) then apply the below styles
  • if only an id with the name container is available, with out any span element then no style will be applied
  • this is just like an AND condition in truth table, when both are satisfied then apply the style else style will not be applied
