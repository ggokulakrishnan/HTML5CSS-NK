CSS 3 Introduction

  1. CSS contains rules for style, in simple terms CSS is used to make the HTML less ugly
  2. In HTML we used to call them as HTML tags but in CSS we used to call them as selectors
  3. The below is an example of CSS tag
    h1 {
      font-size: 24px;
      font-family: Helvetica;
      color: blue;
    }
  4. From the start of h1 to end of } (curly braces) is called a selector
  5. The content written inside the selector are called as properties
  6. h1 is an HTML element, hence it is called HTML selector (or) HTML 5 selector
  7. Conside the below example
    page-settings {
      font-size: 28px;
      color: yellow;
    }
  8. Here the name of the selector is: page-settings and this is called as class Selector
  9. Here (font-size) is the property and its value is 28, where as in HTML 5 it is called as attribute and value
  10. A property should always end with colon (:) and value should always end with semi-colon (;)
  11. (;) indicates that the commend is completed

Picels, Percentage, Points and Ems

  1. Points is rarely used as it mainly used for printing purpose
  2. Example
    h1 {
      font-size: 24px;
      width: 50%;
      padding: 2em;
    }
  3. em: is also its full and short form, consider M of english letter the size of em is equal to the width of one capital M letter
  4. Calculation: 1 em is equal to 16 px by default
  5. in the above example we have given font-size as 24px; then the padding size is 24 X 2 = 48px (here 1 em is 24px)
  6. if the css property font-size as 24px; is not declared then the size of em is 32px; (here 1 em is 16px)
  7. so, em depends on font-size property
  8. In responsive design px are not used but (%) is used, for any specific HTML element where size need to be same then (px) is used
